# The free-time API

Heeft u vrije tijd maar weet u niet hoe deze nuttig te spenderen?
Voelt u zich verveeld maar wilt u terug iets actiefs doen?

De free-time API biedt u suggesties aan over welke activiteiten mogelijk zijn indien u vrije tijd heeft maar geen enkel idee heeft wat met die vrije tijd te doen. U heeft de mogelijkheid om te filteren op zowel welk type activiteit alsook met hoeveel personen u een activiteit wilt doen.
