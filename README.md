# The free-time API

Heeft u vrije tijd maar weet u niet hoe deze nuttig te spenderen?
Voelt u zich verveeld maar wilt u terug iets actiefs doen?

De free-time API biedt u suggesties aan over welke activiteiten mogelijk zijn indien u vrije tijd heeft maar geen enkel idee heeft wat met die vrije tijd te doen. U heeft de mogelijkheid om te filteren op zowel welk type activiteit alsook met hoeveel personen u een activiteit wilt doen. De mogelijkheid is er ook dat als u twijfels heeft over een bepaalde activiteit de API ook voor u kan kiezen of u aan de bepaalde activiteit deelneemt of niet deelneemt. U kunt deze functie ook gebruiken bij andere omstandigheden waar een beslissing genomen moet worden. Tot slot voorziet de API iets wanneer u een pauze nodig heeft welk een klein maar tof gebaar is.
